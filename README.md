# 🏦 Client Management System (C++ File Handling Project)

## 📌 Description

A **console-based client management system** built using **C++** with file handling.

The system allows managing client records stored in a text file database.

## ✨ Features

* 📋 Display all clients list
* ➕ Add new client
* ❌ Delete client (soft delete logic)
* ✏️ Update client information
* 🔍 Search client by account number
* 📂 File-based persistent storage
* 🧾 Structured client record format

## 🛠️ Technical Concepts Used

* Struct-based data modeling
* Vector container storage
* File handling using fstream
* String parsing and serialization
* Menu-driven program architecture

## 📂 Project Structure

* `main.cpp` → Contains full application logic
* `Clients.txt` → Stores client records

## 🚀 How to Run

Clone the repository from GitHub then compile:

```bash
g++ main.cpp -o clientsystem
```

Run the program:

```bash
./clientsystem
```

## 🎯 Usage

1. Start the program.
2. Choose an option from the main menu.
3. Follow on-screen instructions.
